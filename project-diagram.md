# Блок-схема проекта "Эзотерические услуги"

## Структура проекта

```
project/
├── index.html              # Главная страница
├── about.html             # Страница "О нас"
├── catalog.html           # Каталог услуг
├── contacts.html          # Контакты
│
├── styles/                # Стили
│   ├── normalize.css     # Сброс стилей
│   └── style.css         # Основные стили
│
├── scripts/              # JavaScript файлы
│   ├── main.js          # Управление музыкой
│   └── script.js        # Основная логика
│
├── images/              # Изображения
│   ├── logo.jpg        # Логотип
│   ├── favicon.ico     # Иконка сайта
│   └── partner-*.jpg   # Изображения партнеров
│
├── audios/             # Аудио файлы
│   └── background-music.mp3
│
└── fonts/             # Шрифты
```

## Функциональная схема

```
[Пользователь]
    │
    ├── Навигация
    │   ├── Главная
    │   ├── Каталог
    │   ├── О нас
    │   └── Контакты
    │
    ├── Аутентификация
    │   ├── Вход
    │   └── Регистрация
    │
    ├── Услуги
    │   ├── Просмотр каталога
    │   ├── Детальная информация
    │   └── Запись на услугу
    │
    └── Дополнительно
        ├── Управление музыкой
        └── Прокрутка вверх
```

## Технический стек

```
[Frontend]
    │
    ├── HTML5
    │   ├── Семантическая разметка
    │   └── Формы
    │
    ├── CSS3
    │   ├── Grid/Flexbox
    │   ├── CSS Variables
    │   └── Анимации
    │
    └── JavaScript
        ├── Модальные окна
        ├── Валидация форм
        └── Управление аудио
```

## Адаптивность

```
[Разрешения]
    │
    ├── Десктоп (≥1440px)
    │   ├── Трехколоночная сетка
    │   └── Полноразмерное отображение
    │
    └── Планшет (≤768px)
        ├── Двухколоночная сетка
        └── Компактное меню
```

## Взаимодействие компонентов

    ├── [Музыкальный плеер]
